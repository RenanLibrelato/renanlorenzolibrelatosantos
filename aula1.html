Aula de programação front end - 23/02


A tag html é utilizadda para informar ao navegador que tipo de estrutura está sendo construída, podendo ter títulos, imagens, links, entre outros.

Existem diversas tags do HTML, mas as principais são: title, meta, body, footer e div.

Dentro dessa tag, é necessário declarar outras duas tags: head e body.
